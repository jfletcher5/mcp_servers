Data Flow & Process

Understanding how requests flow through the system from input to response.

1. User Input Processing

User submits natural language request via CLI, web interface, or direct API call. Input is validated and prepared for processing.

2. LangGraph Agent Analysis

GPT-powered ReAct agent analyzes the request, determines intent, and selects appropriate tools from the available MCP servers.

3. Reliable MCP Server Communication

Agent communicates with self-healing MCP servers (Task DB via JSON-RPC/STDIO, Notifications via REST/HTTP) with automatic retry and timeout protection.

4. Data Processing & Storage

MCP servers process the requests, update in-memory storage (or databases in production), and return structured responses.

5. Response Generation

Agent processes server responses and generates human-friendly, contextual responses that confirm actions and provide helpful information.

6. User Feedback

Final response is delivered to the user through their chosen interface, completing the request-response cycle.

Improved Communication with Auto-Recovery
async def _call_task_db_server(self, method: str, params: Dict = None) -> str:
    """Communicate with Task DB Server with auto-restart capability."""
    if not self.task_db_process or self.task_db_process.poll() is not None:
        await self._restart_task_db_server()  # Auto-restart if needed
    
    request = {
        "jsonrpc": "2.0", "id": 1, "method": "tools/call",
        "params": {"name": method, "arguments": params or {}}
    }
    
    try:
        # Send with timeout protection
        self.task_db_process.stdin.write(json.dumps(request) + "\n")
        self.task_db_process.stdin.flush()
        
        # Read response with 5-second timeout
        response_line = await asyncio.wait_for(
            self._read_response(), timeout=5.0
        )
        return self._parse_response(response_line)
    except Exception as e:
        await self._restart_task_db_server()  # Auto-recovery
        return f"❌ Communication error: {str(e)}"
